# NSS

**Клиент-серверное приложение маркетплейса** (аналог сервисов вроде Wildberries) на C++. Реализует полный цикл онлайн-заказа: работа с товарами, оформление заказов и выдача в пунктах выдачи.

> Командный учебный проект (НГТУ им. Р. Е. Алексеева). Моя роль — **руководитель разработки**: архитектура, распределение задач, code review и реализация серверной части.

## Возможности
- 🖥️ Клиент-серверное взаимодействие через **сокеты**
- 👤 Управление клиентами
- 📦 Каталог товаров
- 🛒 Оформление и обработка заказов
- 🏪 Пункты выдачи заказов (order pickup points)
- 💾 Хранение данных в **SQLite**

## Архитектура
```
ProgramFiles/NSS
├── src         // бизнес-логика: clients, product, order, order_pickup_point
├── server      // серверная часть + работа с БД (SQLite)
├── socket      // сетевое взаимодействие клиент ↔ сервер
├── namespace   // вспомогательные функции управления
└── include     // заголовочные файлы (интерфейсы сущностей)
```

## Технологии
- **C++**
- **Сокеты** (клиент-серверное взаимодействие)
- **SQLite** — хранилище данных
- Сборка: **CMake** + **vcpkg**

## Как собрать
```bash
git clone https://github.com/NaysasEnterprise/NSS.git
cd NSS
cmake -B build -S . -DCMAKE_TOOLCHAIN_FILE=<путь к vcpkg>/scripts/buildsystems/vcpkg.cmake
cmake --build build
```
Зависимости подтягиваются через vcpkg (см. `vcpkg-configuration.json`).
